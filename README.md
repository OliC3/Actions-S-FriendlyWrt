# Сборка FriendlyWrt для NanoPi R2S Plus через GitHub Actions

## ⚠️ Важная информация

В первую очередь этот форк сделан для меня самого, чтобы в дальнейшем собирать актуальную версию прошивки для **NanoPi R2S Plus**, очищенную от предустановленных языковых пакетов и сервисов. Прошивка не тестировалась на R2S/R2C и работоспособность не гарантируется даже при их совместимости, поэтому будьте осторожнее!

Репозиторий будет переведен в архив после того как версия Plus получит официальную поддержку OpenWrt.
